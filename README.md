Bike Anjo
=========

install:
--------

* create a virtualenv
* (sudo) npm install -g grunt-cli bower
* make

run:
----

you need to run both the django process and grunt to build static assets:

* ./bikeanjo.py
* grunt

then you can open [http://localhost:8000](http://localhost:8000) on your browser.

tests:
----

* make tests
